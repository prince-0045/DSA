
# Leetcode 498: Diagonal Traverse

## Problem Statement

Given an `m x n` matrix `mat`, return *all elements of the matrix in a diagonal order*.

[Leetcode Link](https://leetcode.com/problems/diagonal-traverse/)

---

## âœ… C++ Code

````cpp
class Solution {
public:
    vector<int> findDiagonalOrder(vector<vector<int>>& mat) {
        int m = mat.size();
        int n = mat[0].size();
        vector<int> d;
        int i=0, j=0;
        int q=0;

        while(d.size() < m*n){
            d.push_back(mat[i][j]);
            if(q==0){
                if(j==n-1){
                    i++;
                    q=1;                    
                }
                else if(i==0){
                    j++;
                    q=1;
                }
                else{
                    i--; j++;
                }
            }
            else {
                if(i==m-1){
                    j++;
                    q=0;
                }
                else if(j==0){
                    i++;
                    q=0;
                }
                else{
                    i++; j--;
                }
            }
        }
        return d;
    }
};
````

- ðŸ”¸ `if (j == n - 1)` â†’ At **last column**, can't go further right.

  - Change direction: `q = 1`

- ðŸ”¸ `else if (i == 0)` â†’ At **first row**, can't go further up.

- ðŸ”¸ `else` â†’ Not at boundary.
  - Move diagonally **up-right**: `i--, j++`
